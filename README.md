An advanced, intelligent medical imaging system designed to automate the analysis of Brain MRI scans. The system utilizes Deep Learning techniques to provide accurate detection, classification, and segmentation of brain tumors, serving as a reliable clinical decision support tool for radiologists.

## 🚀 Key Features
* **Automated Classification:** Classifies MRI images into four distinct categories: Glioma, Meningioma, Pituitary, or No Tumor using Transfer Learning.
* **Tumor Segmentation:** Leverages the **U-Net** architecture to precisely delineate tumor boundaries and calculate tumor volume.
* **Image Preprocessing:** Includes digital image processing pipelines (contrast enhancement and noise reduction using N4ITK bias correction).
* **Web-Based Interface:** A user-friendly web application allowing medical professionals to upload scans, run AI inference, and instantly export comprehensive PDF diagnostic reports.

## 🛠️ Tech Stack
* **Deep Learning Frameworks:** Python, TensorFlow, Keras, OpenCV
* **Backend & Database:** Python, SQLite
* **Dataset:** Brain Tumor MRI Dataset (Kaggle)

## 📊 Performance Metrics
* **Overall Classification Accuracy:** 96.2%
* **Sensitivity:** 95.8%
* **Specificity:** 97.1%
